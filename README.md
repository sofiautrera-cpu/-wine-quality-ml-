# “wine-quality-ml”
Clasificación básica / Predicción de Calidad del Vino (Core)
Versión final del proyecto de Machine Learning para la clasificación de calidad de vinos tintos.

Incluye:
- EDA completo
- Preparación del dataset y creación del target binario
- División estratificada en train/test
- Entrenamiento de 4 modelos (KNN, DT, RF, LogReg)
- Evaluación con Accuracy, Precision, Recall, F1 y AUC
- Matrices de confusión y curvas ROC
- Ajuste de umbral (threshold tuning)
- Selección del mejor modelo: Random Forest (200 árboles)
- Análisis comparativo y conclusiones finales

Esta versión consolida el flujo completo de modelado y evaluación.
